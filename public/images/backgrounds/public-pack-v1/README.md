# HTBF Public Background Pack V1

This folder contains approved public background assets used throughout HTBF.

Naming convention:

01-faith-streams
02-worship-light
03-prayer-room
04-testimony-glow
05-deliverance-dawn

Guidelines:
- Use lowercase filenames.
- Use hyphens instead of spaces.
- Prefer WebP when possible.
- Do not rename existing assets without updating references.
